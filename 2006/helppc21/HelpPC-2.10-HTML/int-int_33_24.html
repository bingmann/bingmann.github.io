           INT 33,24 - Get Driver Version, Mouse Type & IRQ Number

       AX = 24h


       on return:
       BH = major version (see below)
       BL = minor version (see below)
       CH = mouse type:
          = 1  bus mouse
          = 2  serial mouse
          = 3  InPort mouse
          = 4  PS/2  mouse
          = 5  Hewlett Packard mouse
       CL = IRQ number:
          = 0  PS/2
          = 2  IRQ 2
          = 5  IRQ 5
          = 7  IRQ 7

       - version 6.1 would be represented as BH = 06h, BL = 10h
