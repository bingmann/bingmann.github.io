                     INT 27 - Terminate and Stay Resident


       DX = offset of last byte in program to remain resident plus 1
       CS = segment of PSP


       returns nothing


       - terminates process without closing open files
       - memory beyond the offset in DX is freed


       - see   INT 21,31
