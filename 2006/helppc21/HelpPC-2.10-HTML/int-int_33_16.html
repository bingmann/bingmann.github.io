                     INT 33,16 - Save Mouse Driver State


       AX = 16h
       ES:DX = far pointer to mouse state save buffer


       returns nothing


       - used to save mouse information before relinquishing control
         to another programs mouse handler
       - see INT 33,15   INT 33,17
