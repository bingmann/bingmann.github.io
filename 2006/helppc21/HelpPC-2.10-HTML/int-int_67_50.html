         INT 67,50 - Map/Unmap Multiple Handle Pages  (LIM EMS 4.0+)


       AH = 50h
       AL = 00 Map/unmap pages
          = 01 map/unmap segments
       DX = EMM handle
       CX = number of entries in array
       DS:SI = pointer to mapping array


       on return:
       AH = 00  success
          = error code (see EMS STATUS)
