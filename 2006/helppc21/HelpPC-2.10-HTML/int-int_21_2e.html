                     INT 21,2E - Set/Reset Verify Switch


       AH = 2E
       AL = 00 to set off
          = 01 to set verify on
       DH = 00 for DOS versions before 3.0


       returns nothing


       - with the verify setting on, disk I/O is more secure but
         takes longer to complete


       - see   INT 21,54
