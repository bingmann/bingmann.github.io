                      TC: void hardresume( int rescode )


       - prototype in dos.h

       - hardresume() may be called with rescode to return to DOS,
         where rescode is:

               2 - abort
               1 - retry
               0 - ignore

       - MS C uses _hardresume()
       - see   harderr()
