                   JNE/JNZ - Jump Not Equal / Jump Not Zero


       Usage:  JNE     label
               JNZ     label
       Modifies flags: None


       Causes execution to branch to "label" if the Zero Flag is clear.
       Unsigned comparision.

                                Clocks                 Size
       Operands         808x  286   386   486          Bytes

       label:  jump      16   7+m   7+m    3            2-4
               no jump   4     3     3     1
